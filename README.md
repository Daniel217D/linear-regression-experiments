# linear-regression-experiments

Несколько ноутбуков по линейным моделям в **scikit-learn**: регрессия с `Pipeline` и кросс-валидацией, разведочный анализ California Housing, классификация текстов (мешок слов и TF‑IDF с логистической регрессией), плюс ноутбук по задаче «черепахи» (Яндекс).

## Зависимости

Python, pandas, numpy, scikit-learn; остальное — по мере нужды внутри ноутбуков.

## Файлы

| № | Ноутбук | Тема |
|---|---------|------|
| 1 | `1. sklearn_pipeline_scaling_cv.ipynb` | Pipeline, масштабирование, CV, Lasso и Ridge |
| 2 | `2. california_housing_eda.ipynb` | EDA California Housing |
| 3 | `3. bag_of_words_with_logistic_regression.ipynb` | BoW и логистическая регрессия |
| 4 | `4. tf_idf_with_logistic_regression.ipynb` | TF‑IDF и логистическая регрессия |
| 5 | `5. yandex-tortilla-turtle.ipynb` | Задача «черепахи» |
